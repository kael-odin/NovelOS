# 第八章摘要

## 本章发生的事

陈萧和林若在孵化场控制室遇到了暗影议会的创始人——陈萧的父亲。他揭示了病毒是进化的筛选器，目的是创造新人类。他还透露陈萧的时间感知能力是他实验的结果。陈父启动了基地的自毁程序，陈萧和林若被迫逃离。孵化场被毁，阿杰牺牲。

## 关键事件

1. 陈萧和林若在控制室遭遇伏击
2. 暗影议会创始人现身，揭示自己是陈萧的父亲
3. 陈父揭示病毒是筛选器，陈萧的能力是实验结果
4. 陈父启动自毁程序
5. 陈萧能力觉醒，击败控制室的敌人
6. 孵化场毁灭，阿杰确认死亡

## 人物状态变化

- 陈萧：得知父亲是暗影议会创始人，能力觉醒
- 陈父：在基地自毁中死亡
- 阿杰：确认死亡

## 推进的伏笔

- 病毒的真相：病毒是筛选器，目的是创造新人类
- 陈萧的过去：陈萧是暗影议会创始人的儿子，能力是实验结果
- 陈萧的母亲：在病毒爆发时死亡，拒绝进化

## 章末状态

陈萧和林若站在废墟外，决定联合更多力量对抗暗影议会。
