# 第4章摘要

## 本章发生了什么

黑衣人揭示身份——他是陈萧曾经的同事李明，暗影议会成员。李明告诉陈萧病毒是暗影议会推动人类进化的计划。两人交手，林若用治愈能力攻击李明，帮助陈萧逃脱。两人到达西区幸存者营地，首领张强认出陈萧——一年前陈萧曾救过他的命。

## 关键事件

1. 李明揭示身份，是暗影议会成员
2. 陈萧与李明交手
3. 林若用治愈能力攻击李明
4. 两人到达幸存者营地
5. 张强认出陈萧

- 发现敌人是曾经的同事
- 到达安全区
- 过去被揭开一角

### 林若
- 展示治愈能力的攻击性
- 对陈萧更加信任
