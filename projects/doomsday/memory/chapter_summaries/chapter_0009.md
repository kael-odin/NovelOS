# 第九章摘要

## 本章发生的事

陈萧和张强分别前往其他营地寻求联盟。陈萧成功说服了北边的"铁壁"营地首领赵刚加入。张强说服了东边的"希望"营地首领李雪加入。当陈萧和李雪会面时，遭到暗影议会的伏击，但两人联手击退了敌人。联盟正式成立。

## 关键事件

1. 陈萧前往铁壁营地，说服赵刚加入联盟
2. 张强前往希望营地，说服李雪加入联盟
3. 陈萧和李雪会面，遭到暗影议会伏击
4. 两人联手击退敌人
5. 幸存者联盟正式成立

## 人物状态变化

- 赵刚：加入联盟
- 李雪：加入联盟
- 陈萧：成为联盟的核心人物

## 新增角色

- 赵刚：铁壁营地首领，硬汉，直率
- 李雪：希望营地首领，聪明，冷静，有野心

## 章末状态

三个营地结成联盟，准备对抗暗影议会。
