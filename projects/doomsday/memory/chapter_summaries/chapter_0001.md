# 第1章摘要

## 本章发生了什么

末日历001年春，病毒爆发一周年。陈萧在废弃公寓醒来，外出搜寻物资时发现一张传单，指向西区研究所安全区。他前往研究所，发现曾经工作的地方已成废墟。在研究所深处，他发现一扇被锁住的铁门，门后传来微弱的呼吸声。

## 关键事件

1. 陈萧在废墟中醒来，清点物资
2. 外出搜寻，遭遇丧尸群
3. 发现传单，得知西区研究所有安全区
4. 前往研究所，发现曾经的实验室
5. 发现被锁住的房间，有人活着

## 人物状态变化

### 陈萧
- 位置：废弃公寓 → 西区研究所
- 资源：水、压缩饼干、罐头、绷带
- 情绪：麻木但有底线

## 伏笔推进

- F001 病毒的真相：发现实验室废墟中的文件碎片
- F002 陈萧的过去：看到实验室时表情异常

## 章末状态

- 陈萧发现铁门后有幸存者
- 准备打开门
