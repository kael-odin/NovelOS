# 第6章摘要

## 本章发生了什么

陈萧调查内鬼，发现是营地副首领王刚。王刚计划在当晚打开大门让暗影议会攻击。陈萧和张强将计就计，设下埋伏。暗影议会发动攻击，但被营地幸存者击退。战斗持续一夜，营地损失惨重但屹立不倒。

## 关键事件

1. 陈萧调查内鬼
2. 发现内鬼是王刚
3. 将计就计设埋伏
4. 暗影议会攻击营地
5. 幸存者击退敌人

- 展示领导力
