# 第5章摘要

## 本章发生了什么

张强讲述一年前被陈萧救出的往事。陈萧和林若融入营地生活，林若成为营地医生，陈萧加入搜寻小队。第五天晚上，陈萧发现暗影议会的信使，得知营地内有内鬼。陈萧决定主动出击，不再逃避。

## 关键事件

1. 张强讲述被救往事
2. 两人融入营地生活
3. 发现暗影议会信使
4. 得知营地有内鬼
5. 决定主动出击

## 人物状态变化

### 陈萧
- 位置：幸存者营地
- 情绪：从逃避到面对
- 决定：主动出击

### 林若
- 成为营地医生
- 地位提升
