# 第1章摘要

## 本章发生了什么

林墨在青云宗外门考核中因修为仅达炼气三层而被判定不合格，本应被逐出外门，但因左臂旧伤被特许留在杂役院。他被分配到杂役院最边缘的石屋，每日需要劈柴担水。深夜，林墨独自在后山修炼时，父亲留下的神秘玉佩突然异动，释放出一股奇异力量，帮助他突破瓶颈，修为从炼气三层直接提升到炼气五层。

## 关键事件

1. 林墨外门考核失败，修为仅炼气三层
2. 被分配到杂役院，地位一落千丈
3. 夜晚独自修炼，玉佩首次异动
4. 修为连破两层，达到炼气五层

## 人物状态变化

### 林墨
- 境界：炼气三层 → 炼气五层
- 位置：青云宗外门 → 杂役院
- 情绪：压抑苦涩 → 震惊 → 重燃希望
- 身体：左臂旧伤仍在

## 伏笔推进

- F001 神秘玉佩的异动：首次激活，释放力量帮助林墨突破，暗示与父亲失踪有关

## 章末状态

- 林墨修为突破到炼气五层
- 玉佩恢复平静，但似乎有什么东西在其中沉睡
- 林墨决心有朝一日进入内门
