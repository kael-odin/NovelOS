# 第3章摘要

## 本章发生了什么

林墨在管事房巧妙化解赵虎的指控，周管事没有惩罚他。深夜，赵虎带人在后山伏击林墨，却被林墨暗中击败。林墨从赵虎身上搜到回气丹、灵石和一块血煞门令牌，怀疑赵虎是血煞门奸细。林墨决定不动声色，继续观察。

## 关键事件

1. 管事房对峙，林墨化解危机
2. 赵虎后山伏击失败
3. 林墨击败赵虎，搜到血煞门令牌
4. 发现赵虎可能是奸细

## 人物状态变化

### 林墨
- 获得：回气丹、灵石×5、血煞门令牌
- 发现：赵虎可能是血煞门奸细

### 赵虎
- 状态：受伤，对林墨产生恐惧

## 伏笔推进

- 新增伏笔：血煞门奸细

## 章末状态

- 林墨继续隐藏实力
- 赵虎受创
- 血煞门阴谋浮出水面
