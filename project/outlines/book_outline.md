# 总纲

> 项目：示例玄幻小说 | 题材：玄幻/成长/热血

## 一、核心主题

**从边缘到中心**：一个被命运抛弃的少年，如何在残酷的修仙世界中找到自己的位置，最终成长为能够守护所爱之人的强者。

## 二、主线冲突

- **表层冲突**：林墨如何在宗门中生存、突破、崛起
- **深层冲突**：林墨与父亲失踪真相的纠葛
- **终极冲突**：林墨与隐藏在幕后的巨大阴谋的对决

## 三、主角成长主轴

1. **第一阶段（1-100章）**：生存与觉醒
   - 从杂役到外门弟子
   - 发现玉佩的秘密
   - 初步建立人际关系

2. **第二阶段（101-300章）**：成长与挑战
   - 内门竞争
   - 宗门大比
   - 遗迹探险

3. **第三阶段（301-500章）**：真相与抉择
   - 发现父亲线索
   - 面对身份真相
   - 做出重大抉择

4. **第四阶段（501-800章）**：巅峰与守护
   - 成为宗门核心
   - 对抗终极敌人
   - 守护所爱之人
